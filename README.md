# NODE TEMPLATE

Full-stack admin starter: **Express + MySQL + React/Vite/Tailwind**.

## Quick start

```bash
cp .env.example .env
npm install
npm run server:dev   # terminal 1
npm run dev          # terminal 2
```

Admin: http://localhost:5173/admin/login (`admin@template.dev` / `admin123`)
